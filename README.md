# FAO_COLOMBIA
Monitoreo de Sistemas Agroalimentarios con cobertura nacional y para Atlántico y Santander. Integra los elabónes de los SA en un tablero web. Usa indicadores trazables para FAO, Alcaldías, Gobernaciones y hacedores de política, permitiendo detectar anomalías, priorizar intervenciones y evaluar planes. Metodología reproducible e interoperable.
